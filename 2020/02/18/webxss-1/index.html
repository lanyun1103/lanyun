1-7信息搜集：

服务器系统，数据库系统，中间件系统，边界设备

收集针对系统管理员的信息收集

方法：

1.主动

直接与目标进行交互，对交互过程中的信息进行收集。（Nmap扫描，Bursuite站点地图收集）

2.被动

通过第三方引擎（google hack，shodan）（不直接交互

shodan：

搜索互联网上的所有设备 shodan.io

指定搜索：

port指定具体端口探测

host指定具体IP地址探测指定IP地址主机信息

city指定搜索具体城市的内容

命令行使用：

安装：

easy_install shodan

获取帮助：shodan -h

初始化：shodan init [apikey]

  • inurl：域名中包含的内容；
  • intitle：标题中包含的内容；
  • intext：页面正文的内容以及代码内的内容；
  • filetype：搜索指定的类型（东侧、txt等）；
  • site：限制搜索的域名；查找主站的子域名

获取ip信息：shodan host [ip]

自身账户信息：shodan info

自身ip：shodan myip

python-shodan使用：

在命令行中打开python

shodan帮助：developer.shodan.io/api
