2-4基于报错的SQL注入

类型

根据“注入位置”数据类型类型：

  • 数字型

    select * from table where id = 用户输入ID

  • 字符型

    select * from table where id = “用户输入ID”

通过在URL中修改对应的ID值：可以是正常数字、大数字、字符（单引号，双引号，双单引号，括号），可以用反斜杠来探测是否存在注入点
