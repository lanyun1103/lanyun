堆排序

发表于 2019-11-05 分类于 文字
堆的性质：

​ 每个结点的值都大于或等于其左右孩子结点的值，称为大顶堆；

​ 每个结点的值都小于或等于其左右孩子结点的值，称为小顶堆。

​ 大顶堆：arr[i] >= arr[2i+1] && arr[i] >= arr[2i+2]

​ 小顶堆：arr[i] <= arr[2i+1] && arr[i] <= arr[2i+2]

堆排序的基本思想是

堆排序的基本思想是：将待排序序列构造成一个大顶堆，整个序列的最大值是堆的根节点。将其与末尾元素进行交换，此时末尾就为最大值。然后将剩余n-1个元素重新构造成一个堆，得到一个n-1个元素，n-1个元素比堆尾（坐标n-1）的值都小。如此反复执行，得到一个有序序列

大顶堆：升序（最后一个为最大

小顶堆：降序（最后一个为最小

步骤
  1. 从最后一个非叶子结点开始（叶结点不调整，第一个非叶子结点 arr.length/2-1），建立最大（最小）堆。
  2. 将根节点的值与最后一节结点的值互换（这样最后的结点就是最大值）
  3. 把除最后一个结点以外的结点，重新构造成一个最大堆
  4. 重复2,3，直到有序
总结：

​ a.将无需序列构建成一个堆，根据升序降序需求选择大顶堆或小顶堆;

　　b.将堆顶元素与末尾元素交换，将最大元素”沉”到数组末端;

　　c.重新调整结构，使其满足堆定义，然后继续交换堆顶元素与当前末尾元素，反复执行调整+交换步骤，直到整个序列有序。

堆排序的优点：

堆排序可以快速找到逆序对较多的数中，最大或者最小的值

时间复杂度：O（nlogn）
空间复杂度：1
